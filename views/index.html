API Project: URL Shortener Microservice

Working

  1. Paste the URL you want to shorten in the form in the above webpage.
    When submitted, the form sends a POST request to the /api/shorturl/new endpoint.
  2. A valid URL will get JSON object as a response which has a code (id), replace it in the :id part of this url:
    https://raahelpie-url-shortener.glitch.me/api/shorturl/:id/
  3. If the URL is already present in the database, it will return the id corresponding to it,
    else - it will create a new document for it and assign a new id.

Short URL Creation

example: POST [project_url]/api/shorturl/new - https://www.google.com

[this_project_url]/api/shorturl/5
